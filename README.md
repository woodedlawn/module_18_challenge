# Blockchain-based Ledger System (module_18_challenge)

A python app that demostrates a blockchain-based ledger system with a web-based ui.

---

## Technologies

- python 3.7.10
- pandas 1.2.4
- streamlit 0.84.2
- typing_extensions 3.7.4.3
